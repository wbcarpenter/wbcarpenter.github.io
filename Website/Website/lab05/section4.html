Constructing a LAN Using One Router

LAN Setup:

To set up an LAN, we each needed a computer, ethernet cable, and an ethernet adapter, along with one router with its modem cable. We started by connecting our computers to the router with our ethernet cables, and plugging the modem cable in. We went through the document configuration and IP assignment process (which I explain more below), then made sure to switch both are computers to the local network.



We were both able to browse each other's Samba shares once we were both connected to the LAN. I included that screenshot here:



TCP/IP Configuration:

After resetting the router, we were able to access the router's configuration web pages. We then selected a new IP range; group 2, 192.168.101.0-1.92.168.101.255.



Data Transmission Experiment:

Once we both had acces to the shared public folder, we went through the process of transferring a file. We each created a 4GB dummy file and dropped it into the shared folder. Mine successfully shared, but Annaliese's did not. We spent time trying to figure out why, and I finally realized that she had entered a different path in the smb.conf file than what it should have been. Once that issue was fixed, she was able to transfer the file to me. The transfer took 15 minutes for both of us. Below are our transmission rates on the System Monitor.
